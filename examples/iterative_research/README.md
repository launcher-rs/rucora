# 迭代深化研究示例 (iterative_research)

## 原理

迭代深化研究示例展示了如何通过多轮迭代逐步深化研究。每轮基于前一轮的发现进行更深入的分析，模拟人类研究的迭代过程。

## 核心组件

- **ToolAgent**: 带工具调用能力的智能体
- **多轮循环**: 每轮将前一轮的研究结果作为上下文
- **上下文累积**: 后续轮次可以看到并利用之前的所有发现

## 迭代流程

1. **第一轮**: 基础信息收集，生成初始摘要
2. **后续轮次**: 基于前轮摘要进行深入挖掘，补充新角度
3. **最终输出**: 累积所有轮次的发现，形成完整报告

## 意义

1. **深度研究**: 支持复杂主题的多层次探索
2. **动态调整**: 根据前轮结果动态调整研究方向
3. **成本控制**: 通过轮次控制预算和深度

## 运行

```bash
cargo run --example iterative_research
```

## 环境变量

需要设置以下环境变量：
- `OPENAI_API_KEY`: OpenAI API 密钥
- `TAVILY_API_KEYS`: Tavily 搜索 API 密钥

## 配置

可通过修改代码中的 `DEFAULT_ITERATIONS` 变量调整迭代轮数。
