# 学术研究示例 (rucora-deep-research-academic)

## 原理

学术研究示例展示了如何针对学术场景进行专业化研究。重点关注学术论文的检索、引用格式和学术规范。

## 核心组件

- **ResearchReport**: 研究报告结构
- **Citation**: 引用条目
- **SourceType::Academic**: 学术来源类型

## 特点

1. **学术来源**: 优先使用 arXiv、PubMed、Semantic Scholar 等学术数据库
2. **规范引用**: 生成符合学术规范的引用格式（APA、MLA 等）
3. **深度分析**: 关注研究方法、实验结果等学术细节

## 报告结构

- **摘要**: 研究主题的简要概述
- **引言**: 研究背景和动机
- **研究方法**: 使用的方法和数据来源
- **主要发现**: 核心研究成果
- **结论**: 研究意义和局限性
- **参考文献**: 规范格式的引用列表

## 意义

1. **学术场景**: 适合论文写作、文献综述
2. **引用规范**: 自动生成符合规范的引用
3. **可验证性**: 引用可追溯到原始学术来源

## 运行

```bash
cargo run --example rucora-deep-research-academic
```

## 扩展

可以集成 ArxivTool、PubMedTool 等专门的学术搜索工具来自动获取真实的学术论文信息。
